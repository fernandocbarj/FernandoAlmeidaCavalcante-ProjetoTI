# Segurança da Informação

A segurança da informação é tratada como elemento central desta arquitetura, considerando os riscos inerentes a ambientes institucionais.

## Controles adotados
- Isolamento de aplicações
- Uso de HTTPS em todos os serviços
- Redução da superfície de ataque
- Separação entre camada de aplicação e dados

## Conformidade
A solução está alinhada conceitualmente aos seguintes referenciais:
- Lei Geral de Proteção de Dados (LGPD – art. 46)
- ISO/IEC 27001
- NIST Cybersecurity Framework
